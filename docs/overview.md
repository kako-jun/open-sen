# open-sen とは

## 生まれた動機

何かを世に出した人には、共通する悩みがある。

「宣伝してみたけど、手応えがわからない」

Zenn に記事を書いて、Reddit にも投稿して、GitHub の Star も気になる。複数のサイトを行き来して数字を確認する。でも、それぞれの数字がバラバラに存在していて、全体像がつかめない。昨日より増えたのか、減ったのか、どのタイミングで伸びたのか——そういう「推移」が手元に残らない。

そして、反応が薄いとき。数字が増えないとき。それだけで気持ちが折れてしまう。

open-sen は、この体験を変えようとして作られた。宣伝活動の手応えを一か所に集めて可視化し、モチベーションを維持し続けることを助けるダッシュボードである。

## 名前の由来

「open-sen」は、野球の「オープン戦」から来ている。

二つの意味が重なっている。

ひとつは「オープン宣伝」——プロモーション状況をオープンに晒すという姿勢。数字が良くても悪くても、隠さずに外に出すことで、誠実さと透明性を示す。

もうひとつは「オープン戦（戦い）」——有名になれるかどうかの、まだドキドキしている序盤戦。プロモーションとは、世界に向けての挑戦である。結果を恐れずバッターボックスに立つことが大事だ、という精神がこの名前に込められている。

## 野球テーマである理由

プロモーションの結果が芳しくないとき、人は落ち込む。数字を見るのがつらくなる。

でも、プロ野球選手は3割打てば一流と言われる。10打席のうち7回は凡退する。それでも打席に立ち続ける。

宣伝活動もそれと同じだ。全部バズらなくていい。三振しても次の打席に向かえばいい。

open-sen は、野球をテーマにすることで、数字に一喜一憂しすぎない精神的な余裕を持てるように設計されている。励ましのメッセージも、スコアボードのような見た目も、すべてその思想から来ている。

## 対象ユーザー

open-sen は、「何かを世に出したすべての人」のためのツールである。

- OSSツールやNeovimプラグインを公開した開発者
- Zenn・Qiita に技術記事を書いたライター
- インディーゲームやWebサービスを作った個人開発者
- 新店舗をオープンしてSNSで宣伝中のオーナー
- YouTube・Instagram・TikTokで活動するクリエイター

技術者だけのツールではない。発信する人すべてに開かれている。
