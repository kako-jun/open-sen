# 使い方

## ログイン

https://open-sen.llll-ll.com にアクセスし、「Sign in」ボタンをクリックする。

メールアドレスを入力すると、そのアドレス宛に数字のコードが届く。そのコードを入力すれば認証完了である。パスワードの登録は不要。外部サービスのアカウントとの連携も不要。メールアドレスだけで使い始められる。

認証セッションは24時間有効である。翌日以降に再度アクセスする場合は、同じ手順でログインし直す。

## プロジェクトを作る

ログイン後、プロジェクト一覧画面が表示される。

「新規プロジェクト」ボタンから作成画面に進み、プロジェクト名を入力して登録する。名前は自由につけてよい。「chillout.nvim の宣伝」でも「ラーメン屋のInstagram」でも構わない。

GitHub リポジトリのURLを関連付けることもできる。

## 投稿URLを登録する

プロジェクトの詳細画面から、投稿のURLを追加できる。

URLを貼り付けるだけでよい。どのプラットフォームの投稿かは自動的に判別される。Zennの記事URL、GitHubリポジトリURL、RedditのスレッドURL——そのまま貼り付ければ準備完了である。

X（Twitter）やInstagramなど、自動取得に対応していないプラットフォームのURLを登録した場合は、エンゲージメント数の入力欄が表示される。手元で確認した数値を入力する。

## データの見方

プロジェクト詳細画面には、登録した投稿ごとのエンゲージメント数と、時系列グラフが表示される。

グラフの横軸は日付、縦軸はエンゲージメント数である。自動取得対応のプラットフォームは、毎日1回データが更新されるため、しばらく使い続けることで推移が見えてくる。

「この記事を書いた後に Star が増えた」「この宣伝投稿がきっかけだった」——そういった気づきをグラフから読み取ることができる。

登録直後や数値が少ない時期は、野球テーマの励ましメッセージが表示される。まだデータが少なくても心配しなくてよい。積み重ねていくことが大事である。

## 公開設定

現在開発中の機能として、ダッシュボードを公開URLとして共有できる設定がある。

公開すると、ログインしていない人でも自分のダッシュボードを閲覧できるようになる。プロモーション状況を透明に見せることも、あえてオープンに晒すことも自分で選べる。この機能は現バージョンでは利用できない。

## ログアウト

ヘッダーのログアウトボタンからセッションを終了できる。
