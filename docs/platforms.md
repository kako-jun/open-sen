# 対応環境

## ブラウザ

モダンブラウザの最新版に対応している。

- Chrome
- Firefox
- Safari
- Edge

スマートフォン・タブレット・PCのいずれからでも利用できる。画面サイズに応じてレイアウトが最適化される。

## 認証方式

メールアドレスによるワンタイムPIN認証を採用している。

メールアドレスを入力すると、そのアドレス宛に数字のコードが届く。コードを入力するだけでログインできる。パスワードの設定は不要。GitHubやGoogleなどの外部サービスとの連携も不要である。

### プライバシーへの配慮

認証の仕組みは外部サービスに委託されており、サービス本体にメールアドレスそのものは渡らない設計になっている。サービス内部で保持するのは、メールアドレスから生成された識別子のみである。

個人情報の最小化を設計方針としており、サービスが保存するデータはプロジェクト名・投稿URL・エンゲージメント数だけである。

## 公式サービスとセルフホスト

### 公式サービス（推奨）

**https://open-sen.llll-ll.com**

アカウント登録不要ですぐに使い始められる。インフラの管理も不要である。

### セルフホスト

ソースコードは公開されており、技術的にはセルフホストも可能である。ただし、公式サービスの利用を主として想定しており、セルフホストは開発者向けのおまけ扱いである。

セルフホストを試みる場合は、Cloudflare のアカウントと、一定の技術的知識が必要になる。サポートは提供していない。

## データ取得の仕様

自動取得対応プラットフォームのデータ収集は、毎日1回（UTC 15:00）のバッチ処理で行われる。リアルタイムの反映はされない。

また、各プラットフォームの仕様変更により、自動取得が停止する可能性がある。その場合でも、手動入力に切り替えることでデータの記録を続けることができる。
