# open-sen

**プロモーション活動を記録・可視化するダッシュボード**

> 野球の「オープン戦」にかけて:
> - **オープン宣伝** - プロモーション状況をオープンに晒す
> - **オープン戦（戦い）** - 有名になれるかの戦いの始まり

**使ってみる**: https://open-sen.llll-ll.com

## こんな人におすすめ

- OSSツールを作ったので宣伝したい開発者
- 新店舗をオープンしてInstagramで宣伝中のオーナー
- YouTube動画の反応を追いたいクリエイター
- 自分の活動がどう受け入れられているか知りたいすべての人

## 使い方

1. GitHubアカウントでログイン
2. プロジェクト（宣伝したいもの）を登録
3. 投稿URLを追加（Zenn記事、Instagram投稿など）
4. 毎日自動でいいね数などを取得
5. グラフで推移を確認

## 対応プラットフォーム

### 自動取得
URLを登録するだけで、いいね数やコメント数を自動取得:

- **Zenn** - 技術記事
- **Qiita** - 技術記事
- **Note** - ブログ・エッセイ
- **Reddit** - 海外向け投稿
- **GitHub** - Star数・Fork数

### 手動登録
エンゲージメント数は手動で入力:

- **X (Twitter)**
- **Instagram**
- **YouTube**
- **TikTok**
- **Facebook**
- **Threads**

## 特徴

### 野球テーマで落ち込まない

プロモーションがうまくいかなくても大丈夫。バッターボックスに立ち続けることが大事。

- 「三振を恐れるな、バットを振れ」
- 「全打席ホームランの選手はいない」
- 「さぁ、次の打席に立て！」

### URLだけで完結

面倒な設定は不要。投稿URLを登録するだけで始められます。

### 結果をシェア

ダッシュボードのURLを公開して、プロモーション状況を「こんな感じです」と晒せます。

## セルフホスト

自分でホストしたい場合は、以下の技術スタックを使用:

- **API**: Cloudflare Workers + Hono
- **DB**: Cloudflare D1
- **Frontend**: Astro + React
- **Hosting**: Cloudflare Pages

詳細は [CLAUDE.md](./CLAUDE.md) を参照。

- 保存されるデータ: プロジェクト名、投稿URL、エンゲージメント数のみ
- 各プラットフォームのAPI仕様変更により、データ取得が停止する可能性があります
